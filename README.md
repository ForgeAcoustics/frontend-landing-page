# ForgeAcoustics Landing Page

A dark, futuristic landing page for ForgeAcoustics, built using the xAI-inspired design system from DESIGN.md.

## Design System

This website follows the xAI design principles:
- Dark background (#1f2228)
- White text (#ffffff)
- Geist Mono for headlines and buttons
- Inter for body text
- Sharp corners, no shadows
- Monospace typography for technical credibility

## Content

The content is based on prompt.md, covering:
- Hero section with main value proposition
- Problem statement about ultrasound AI data limitations
- Solution pipeline visualization
- Platform offerings (Forge Flow and Forge Suite)
- Use cases in medical imaging
- Value propositions for researchers and investors
- Team information
- Call-to-action for booking demos

## Running the Website

1. Start a local HTTP server:
   ```bash
   python3 -m http.server 8000
   ```

2. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Inter, JetBrains Mono)
- Vanilla JavaScript for scroll animations

## Features

- Fully responsive design
- Smooth scroll animations
- Dark theme with high contrast
- Clean, minimal aesthetic
- Mobile-friendly navigation
